# AI-Mental-Health-Chatbot
## Description

This project is an AI-powered chatbot that provides emotional support by analyzing user input using Natural Language Processing (NLP) and sentiment analysis.

## Technologies Used

- Python / Node.js
- NLP (Natural Language Processing)
- HTML, CSS, JavaScript
- (Optional: MongoDB / Firebase)

## Features

- Interactive chatbot interface
- Sentiment analysis of user input
- AI-based emotional support responses
- Simple and user-friendly design
